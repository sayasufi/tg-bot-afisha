# Окрест — каталог дизайн-идей

Живой бэклог идей по дизайну miniapp (карта событий Москвы, стиль **VITRINE / editorial after-dark**:
плашечный белый куб днём, тёпло-чернильный зал ночью, акцент — акид (день) / янтарь (ночь), один раз — киноварь;
шрифты Unbounded + Familjen Grotesk + Martian Mono; hairline-рамки, crop-метки, моно-подписи).

Статусы: ✅ сделано · ⬜ бэклог · ⭐ приоритет · ❌ отклонено пользователем.

---

## ✅ Уже реализовано
- Пины: live-пульс (киноварь), всплытие выбранного + приглушение прочих, цветовой рейл категории.
- Кластер-тап → мини-список (peek) для стопки в одной точке.
- Метро у события (ближайшая станция + пинг на карте), исключён монорельс.
- Каскад проявления пинов от центра при первой загрузке.
- Карточка: похожие события, кнопка «Маршрут» (пеший Яндекс), мини-карта (была, убрана).
- Фильтры: день-стрип (14 дней), мульти-категория, слайдер радиуса «Рядом».
- Рекомендации: сегмент «Сегодня/Неделя/Всё», секция «Для тебя» (по вкусу).
- Избранное: вкладка + бейдж; профиль «Твой вкус» (диаграмма категорий).
- Движение/состояния: pull-to-refresh, shimmer первой загрузки, live-тикер «идёт сейчас N»,
  расширенные хаптики, брендовый/умный пустой экран.
- Тёмная тема after-dark (акцент → янтарь), переключатель в меню.
- Данные: парсер цен (кириллица/диапазоны), категоризация с подсказками источника + усиленный
  промпт, таксономия 8→11 (кино/экскурсии/вечеринки), backfill существующих.
- **Эстетика (волны 1–2, по variant.com):** serif-акцент Instrument Serif + italic (C1/C2),
  моно код-комменты `// …` (C3), числа-герои в профиле (B7/A3/C7), frosted-glass пилюля (C4),
  тёплое свечение под выбранным пином (B1/A13/C8), depth-blur карты за карточкой (A11/B13),
  генеративное dot-поле (C5/A12/B22), кинетический сплэш + гало эмблемы (A1/B2/B21),
  дуотон обложек похожих (B16/A14), darkroom-проявление обложки (A5), тактильные пружинки (A17).
- **Эстетика (волна 3):** режим «контрольный лист» — сетка дуотон-миниатюр с accession-номерами
  (B19/A21/C19), dot-matrix пульс тикера (B4), аврора-стрик на загрузке (B3).
  Отложено как рискованное на Leaflet: кластер-созвездие (B6), FLIP-переход пин→карточка (A6).

## ⬜ Бэклог — глубокий анализ 134 дизайнов variant.com (batch C)

**Данные по 134 элитным дизайнам:** шрифты — Times New Roman ×104 (доминанта), Inter ×79,
моно (JetBrains ×31 / Space Mono ×18 / VT323), Cormorant Garamond ×20, Playfair ×9, Instrument
Serif ×10. Тема: свет 74 / тьма 53. Медиа: **40 canvas** (генеративка), 40 SVG, **49 анимировано**,
0 видео (вся графика — код). Копирайт: «Introducing X», «// VISUAL_OUTPUT_FACILITY», «architecture
of X», ALL-CAPS, моно-комменты. **Вывод:** элитная связка = классический serif-дисплей + моно-лейблы
+ Inter body + генеративный canvas. Наш Unbounded можно дополнить тонким serif-акцентом.

C1. ⭐ **Serif-акцент для особых моментов** — рафинированный serif (Cormorant/Playfair/Instrument)
    для hero-названия события или сплэш-строки, в пару к Unbounded (Times — №1 у топов).
C2. ⭐ **Italic-emphasis в заголовках** — одно слово курсивом в serif-заголовке
    («события *рядом*», «*тихий* вечер») — журнальный приём («woven», «Thought & PAPER MEET»).
C3. ⭐ **Моно код-комменты как системные лейблы** — `// ОКРЕСТ · МОСКВА`, `FIELD_DRAFT_01`;
    усилить accession слэш-синтаксисом (как `// VISUAL_OUTPUT_FACILITY`).
C4. ⭐ **Frosted-glass панели над картой** — командная пилюля / тикер / peek на полупрозрачном
    фросте (rgba + backdrop-blur), как массовые white-overlay у топов. Глубина без тяжести.
C5. ⭐ **Генеративный canvas-слой** — тонкое дышащее точечное/волновое поле на пустом/сплэше/
    загрузке (30% топ-дизайнов — canvas) вместо статичного фона.
C6. **Magazine-masthead секции** — «ГДЕ ИСКУССТВО ВСТРЕЧАЕТ ГОРОД» крупным serif-разворотом
    как заголовок раздела.
C7. **Стрик/стат большим числом** — «42 события рядом», «твой 5-й вечер» (гейм-стат в профиле).
C8. **Старберст-радиал в after-dark** — мягкий радиальный световой градиент из выбранного события.
C9. **Капсулы-объекты категорий** — лёгкие объёмные пилюли-метки в hero/обзоре (как «woven» pills), опц.
C10. **Рассыпанные соцветия-метки** — крошечные ＋/✛ киноварью как акцентный слой (как EVELYN/stephanie).
C11. **Condensed-дисплей над фото** — «SCALE»-момент: огромный сжатый заголовок поверх обложки.
C12. **Stagger-раскрытие сетки** — карточки контрольного листа каскадом (staggered reveal — стандарт).
C13. **Translucent-hairline разделители** — полупрозрачные линии (rgba) вместо сплошных, для воздуха.
C14. **«Field Draft» режим** — черновик-эстетика: моно-лейблы + dashed-рамки + номер кадра (ср. A9).
C15. **Радиальный «explore the area»** — круговой индикатор охвата вокруг тебя на карте.
C16. **Editorial «Introducing»** для еженедельной подборки («Introducing: вечер четверга»).
C17. **Объёмный знак на сплэше** — один 3D-объект/эмблема с медленным вращением (как RxH/капсулы).
C18. **Двухколоночный «paper» лейаут** длинного описания в карточке (газетная полоса).
